贝叶斯决策

贝叶斯定理: 假设有 ccc 个类别(状态) {w1,w2,…,wc}\{w_1, w_2, \dots, w_c\}{w1​,w2​,…,wc​}, 以及观察到的特征向量 xxx

其中,

  • P(wi)P(w_i)P(wi​): 先验概率
  • P(wi∣x)P(w_i|x)P(wi​∣x): 后验概率, 观察到 xxx 后属于类别 wiw_iwi​ 的概率
  • P(x∣wi)P(x|w_i)P(x∣wi​):
  • P(x)P(x)P(x): 证据因子, P(x)=∑j=1cP(x∣wj)P(wj)P(x) = \sum_{j=1}^cP(x|w_j)P(w_j)P(x)=∑j=1c​P(x∣wj​)P(wj​)

为了使分类错误的概率最小, 应该选择后验概率最大的类别. 同理, 贝叶斯分类中, 决策边界由两个类别后验概率相等 (对数判别函数相等) 决定

为什么 CART 算法最流行

  1. 既能做分类, 也能做回归

  2. 构建二叉树, 计算和实现都简单

  3. Gini 基尼指数的计算没有对数运算, 速度比熵快

  4. 有完善的剪枝机制

  5. 是集成学习算法如 XGBoost, GBDT, 随机森林的基石

  6. ID3 决策树: 信息增益 (划分前信息熵 - 划分后的加权平均信息熵), 仅分类, 仅离散. 倾向于选择取值类别较多的特征 (熵的计算决定的)

  7. C4.5 决策树: 信息增益率 (信息增益除以特征固有的信息熵), 仅分类, 离散和连续

  8. CART 决策树: 基尼指数 (随机抽两个样本呢, 类别不一致的概率, Gini 越小纯度越高), 分类和回归, 离散和连续

KNN 和 K-Means

  • KNN: 监督学习, 分类任务
  • K-Means: 无监督学习, 聚类任务
    随机选 K 个中心, 分成 K 个簇, 重新计算每一簇的中心位置, 迭代计算

回归分析

模型的泛化误差可以分解为以下三个部分:

  1. 偏差: 模型期望预测与真实结果间的偏离程度, 反应了模型本身的拟合能力
  2. 方差: 度量了同样大小的不同数据集变动导致的性能变化, 高方差意味着模型过拟合
  3. 噪声: 数据固有属性, 无法消除

高方差、低偏差（如复杂的决策树、神经网络）：用 Bagging (并行)来平滑方差。

高偏差、低方差（如简单的逻辑回归、浅层树）：用 Boosting (串行)来提升拟合能力。

层次聚类法

自底向上:

  1. 计算样本间的距离, 每个样本点开始时是一个独立的聚类
  2. 合并最小距离的聚类
  3. 新的聚类与其他聚类间的距离根据某种准则重新计算
  4. 迭代直至所有点合并为一个聚类
  5. 树状图可以看到聚类过程

特征选择与特征提取

特征选择在降维中的作用

  1. 减少冗余与无关特征
  2. 提升模型效率
  3. 增强模型可解释性
  4. 优化模型性能

特征提取在降维中的作用

  1. 实现数据紧致压缩
  2. 提取本质特征模式
  3. 提升模型泛化能力
  4. 适配多领域应用

二者的主要区别:

  1. 特征选择从 DDD 个特征中选择 ddd 个来表达模式
  2. 特征提取采用特征变换的方法, 生成 ddd 个新的特征

概率图模型

  • 有向图模型: 贝叶斯网络
  • 无向图模型: 马尔科夫随机场

条件独立性的判断:

  • 有向图模型: D-分离准则, 若 A 和 B 的所有无向路径都被 C 阻塞, 则称 A 和 B 被 C D-分离, 即 A 和 B 关于 C 条件独立
    1. 尾对尾: A <- C -> B
    2. 头对尾: A -> C -> B
    3. 头对头: A -> C <- B
  • 无向图模型: 全局马尔科夫性. 去掉 C 后, A 和 B 不再连通, 则称 A 和 B 被 C 分离, 即 A 和 B 关于 C 条件独立 (A⊥B∣CA \perp B | CA⊥B∣C)
