贝叶斯决策

贝叶斯定理: 假设有 $c$ 个类别(状态) ${w_1, w_2, \dots, w_c}$, 以及观察到的特征向量 $x$

其中,

  • $P(w_i)$: 先验概率
  • $P(w_i|x)$: 后验概率, 观察到 $x$ 后属于类别 $w_i$ 的概率
  • $P(x|w_i)$:
  • $P(x)$: 证据因子, $P(x) = \sum_{j=1}^cP(x|w_j)P(w_j)$

为了使分类错误的概率最小, 应该选择后验概率最大的类别. 同理, 贝叶斯分类中, 决策边界由两个类别后验概率相等 (对数判别函数相等) 决定

线性分类

线性分类中真实标签 $y$ 取 +1 或-1, 所以通过线性判别函数 $y(wx+b)$ 的正负来判断分类是否正确

目标函数 (损失函数) 为 $J(\theta)$, 参数是 $\theta$, 学习率 $\eta$

  1. BGD 批量梯度下降: $\theta{t+1} = \theta_t - \eta \cdot \frac{1}{m} \sum{i=1}^m \nabla_{\theta} L(\theta_t;x^{(i)},y^{(i)})$
  2. SGD 随机梯度下降: $\theta{t+1} = \theta_t - \eta \cdot \nabla{\theta}L(\theta_t;x^{(i)},y^{(i)})$

计算步骤

  1. 初始化模型参数 $\theta$ (或是 $w$ 和 $b$)
  2. 遍历每个样本, 计算
    • 前向传播: 当前样本的预测值 $\hat{y}^{(i)}$
    • 计算损失: 计算单点损失 $L^{i}$
    • 反向传播: 计算关于参数的梯度 $\nabla_\theta L^{(i)}$
    • 参数更新: 线性判别函数$y(wx+b)\leq 0$, 则分类错误, 需要更新参数, $\theta\leftarrow\theta-\eta\cdot\mathrm{Gradient}$
  3. 重复, 直至达到停止条件

Fisher 最优判别分析 (LDA)

寻找一个投影方向, 使得类内散度最小, 类间散度最大

  • 类内散度矩阵 $Sw = \sum_i\sum{x\in D_i}(x-\mu_i)(x-\mu_i)$, 衡量类内数据的离散程度, $\mu_i$ 是第 $i$ 类样本的均值向量
  • 类间散度矩阵 (以二分类为例) $S_b = (\mu_1-\mu_2)(\mu_1-\mu_2)^T$
  • Fisher 准则函数: 寻找向量 $w$, 使得瑞利商 $J(w)=\frac{w^TS_bw}{w^TS_ww}$ 最大, 分子是投影后的类间距离, 分母是投影后的类内方差之和
  • $J(w)$ 求导并令其为 0, 解得最优鉴别矢量 $w^$. 二分类的结果为 $w^ = S_w^{-1}(\mu_1-\mu_2)$

计算流程:

  1. 求均值: 计算两类样本的均值向量 $\mu_1$ 和 $\mu_2$
  2. 求类内离散度矩阵 (方差): 计算 $S_w = S_1 +S_2$, $S_i = \sum(x-\mu_i)(x-\mu_i)^T$
  3. 代入 $w^* = S_w^{-1}(\mu_1-\mu_2)$

决策树

  • 信息熵: $H = -\sum_{k=1}^k p_k\log_2p_k$
  • 信息增益: ID3 使用, 表示得知特征 A 的信息后, 数据集 D 的不确定性减少了多少,
  • 信息增益率: C4.5 使用, 引入了”固有值”作为分母进行惩罚

为什么 CART 算法最流行

  1. 既能做分类, 也能做回归
  2. 构建二叉树, 计算和实现都简单
  3. Gini 基尼指数的计算没有对数运算, 速度比熵快
  4. 有完善的剪枝机制
  5. 是集成学习算法如 XGBoost, GBDT, 随机森林的基石
  1. ID3 决策树: 信息增益 (划分前信息熵 - 划分后的加权平均信息熵), 仅分类, 仅离散. 倾向于选择取值类别较多的特征 (熵的计算决定的)
  2. C4.5 决策树: 信息增益率 (信息增益除以特征固有的信息熵), 仅分类, 离散和连续
  3. CART 决策树: 基尼指数 (随机抽两个样本呢, 类别不一致的概率, Gini 越小纯度越高), 分类和回归, 离散和连续

KNN 和 K-Means

  • KNN: 监督学习, 分类任务
  • K-Means: 无监督学习, 聚类任务
    随机选 K 个中心, 分成 K 个簇, 重新计算每一簇的中心位置, 迭代计算

AdaBoost

迭代训练一系列弱分类器, 组合成强分类器

假设有训练集 ${(x_1, y_1), \dots, (x_N, y_N)}$, 标签 $y\in{-1, +1}$

  1. 初始化: 所有样本权重平均分配 $w_{1,i} = \frac{1}{N}$
  2. 迭代训练
    1. 训练弱分类器: 基于当前样本权重分布 $D_m$, 训练弱分类器 $G_m(x)$ (通常使用单层决策树)
    2. 计算误差率 $\epsilonm = \sum{i=1}^N w_{m,i} \cdot I(G_m(x_i) \neq y_i)$, 即错分样本的权重之和
    3. 计算分类器权重 $\alpha_m = \frac{1}{2} \ln \left( \frac{1 - \epsilon_m}{\epsilon_m} \right)$, $\epsilon_m$ 越小说明表现越好
    4. 更新样本权重 (提升分错样本的权重) $w{m+1,i} = \frac{w{m,i}}{Z_m} \exp(-\alpha_m y_i G_m(x_i))$
  3. 构建强分类器: $M$ 个弱分类器加权线性组合 $f(x) = \sum_{m=1}^M \alpha_m G_m(x)$, 分类结果取符号 $G(x) = \text{sign}(f(x))$

回归分析

模型的泛化误差可以分解为以下三个部分:

  1. 偏差: 模型期望预测与真实结果间的偏离程度, 反应了模型本身的拟合能力
  2. 方差: 度量了同样大小的不同数据集变动导致的性能变化, 高方差意味着模型过拟合
  3. 噪声: 数据固有属性, 无法消除

高方差、低偏差（如复杂的决策树、神经网络）：用 Bagging (并行)来平滑方差。

高偏差、低方差（如简单的逻辑回归、浅层树）：用 Boosting (串行)来提升拟合能力。

层次聚类法

自底向上:

  1. 计算样本间的距离, 每个样本点开始时是一个独立的聚类
  2. 合并最小距离的聚类
  3. 新的聚类与其他聚类间的距离根据某种准则重新计算
  4. 迭代直至所有点合并为一个聚类
  5. 树状图可以看到聚类过程

最小二乘估计

找到一组参数, 使得模型预测值与真实观测值间的误差平方和最小

特征选择与特征提取

特征选择在降维中的作用

  1. 减少冗余与无关特征
  2. 提升模型效率
  3. 增强模型可解释性
  4. 优化模型性能

特征提取在降维中的作用

  1. 实现数据紧致压缩
  2. 提取本质特征模式
  3. 提升模型泛化能力
  4. 适配多领域应用

二者的主要区别:

  1. 特征选择从 $D$ 个特征中选择 $d$ 个来表达模式
  2. 特征提取采用特征变换的方法, 生成 $d$ 个新的特征

概率图模型

  • 有向图模型: 贝叶斯网络
  • 无向图模型: 马尔科夫随机场

条件独立性的判断:

  • 有向图模型: D-分离准则, 若 A 和 B 的所有无向路径都被 C 阻塞, 则称 A 和 B 被 C D-分离, 即 A 和 B 关于 C 条件独立
    1. 尾对尾: A <- C -> B
    2. 头对尾: A -> C -> B
    3. 头对头: A -> C <- B
  • 无向图模型: 全局马尔科夫性. 去掉 C 后, A 和 B 不再连通, 则称 A 和 B 被 C 分离, 即 A 和 B 关于 C 条件独立 ($A \perp B | C$)
