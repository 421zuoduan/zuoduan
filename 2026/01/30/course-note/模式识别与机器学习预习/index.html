    • 协同训练 Co-Training: 核心假设是多视图, 数据必须拥有两个充分且独立的特征子集 (特征互相独立, 且任何一个特征子集都能训练出来一个强分类器, 一个特征子集称为一个视图)
      1. 初始化: 基于有标签数据在两个视图上分别训练模型 A, B
      2. A 和 B 对无标签数据预测
      3. 筛选: 两个模型选出置信度最高的几个正例和负例
      4. 互教: 选出的样本加入对方的数据中, 重新训练
      5. 迭代: 扩充有标签数据, 直至用完所有数据
    • 标签传播算法
      • 核心思想: 基于图, 相似的数据点有相同的标签, 一个节点的标签取决于它的邻居们大多数是什么标签
      • 计算步骤
        1. 建图: 所有数据点视为图中节点, 节点间的边表示相似度 (使用 KNN 或 RBF 核函数建图)
        2. 传播-迭代: 节点根据邻节点标签的甲醛投票来更新自己的标签分布, “我属于 A 类的概率 = ∑\sum∑ (邻居 jjj 属于 A 类的概率 ×\times× 我和 jjj 的连接权重)”
        3. 重置/钳制: 传播后, 有真实标签的数据重置回初始状态
        4. 收敛: 重复23, 直至所有节点标签无变化
    • EM 算法是半监督生成式模型 (如半监督朴素贝叶斯)
      • E: expectation 猜标签: 用当前模型预测无标签数据, 返回概率
      • M: Maximization 重新训练: 把猜出来的概率当作事实, 按照权重更新名参数
      • 迭代

有监督学习

L1 正则和 L2正则在损失函数后加一个惩罚项来限制模型参数 www 的大小, 从而控制模型复杂度

  • L1 正则: L=L0+λ∑i∣wi∣L = L_0 + \lambda \sum_{i} |w_i|L=L0​+λ∑i​∣wi​∣
  • L2 正则: L=L0+λ∑iwi2L = L_0 + \lambda \sum_{i} w_i^2L=L0​+λ∑i​wi2​
特性 L1 正则化 (Lasso) L2 正则化 (Ridge)
主要效果 稀疏性 (Sparsity) 权重衰减 (Weight Decay)
参数变化 会将不重要的特征权重压缩为 0。 会将权重压缩得很小（趋近于 0），但几乎不会等于 0。
应用场景 特征选择：当你觉得特征很多，但只有少数几个有用时。 防止过拟合：处理共线性（Collinearity），让模型更平滑、稳定。
求解难度 在 0 点不可导（是一个尖角），需要特殊算法（如坐标下降法）。 处处可导，计算方便（闭式解或梯度下降均可）。
唯一解 在某些情况下可能有多个解。 总是能得到唯一解（增加了目标函数的凸性）。
  • L1 的图形是“菱形” (Diamond Shape):∣w1∣+∣w2∣≤C|w_1| + |w_2| \le C∣w1​∣+∣w2​∣≤C 的图形是一个有尖角的菱形。损失函数的等高线向中心收缩时，很容易先碰到菱形的顶点（角上）。在顶点处，往往意味着某个坐标轴的值为 0（例如 w1=0,w2=Cw_1=0, w_2=Cw1​=0,w2​=C）。这就是为什么 L1 会产生稀疏解（让某些权重变 0）。
  • L2 的图形是“圆形” (Circle Shape):w12+w22≤Cw_1^2 + w_2^2 \le Cw12​+w22​≤C 的图形是一个圆。损失函数的等高线更容易碰到圆周上的任意一点，而不是刚好在坐标轴上。所以 w1w_1w1​ 和 w2w_2w2​ 都会被压缩变小，但很难同时出现某一个变为 0 的情况。

  • 软间隔: 允许少来那个样本错误或位于间隔内

    • 松弛变量 ξ\xiξ: 为每个样本引入一个变量, 量化它未被间隔的程度
    • 惩罚系数 CCC: C 很大, 对错误零容忍, 容易过拟合; C 很小, 追求更宽的间隔, 容易欠拟合

核函数

核函数: 处理非线性. 低维空间无法线性分割时, 使用核函数将数据映射到高维特征空间

常见核函数:

  1. 线性核 K(x,y)=xTyK(x, y) = x^T yK(x,y)=xTy: 适用于特征维度很高, 或数据本身线性可分的情况
  2. 多项式核: 引入多项式特征
  3. 高斯核 RBF Kernel K(x,y)=exp⁡(−γ∣∣x−y∣∣2)K(x, y) = \exp(-\gamma ||x-y||^2)K(x,y)=exp(−γ∣∣x−y∣∣2): 最常用, 可以将数据映射到无穷维空间
    • 参数 gammagammagamma: 控制高斯分布的宽度. γ\gammaγ 大, 附近的点影响大, 决策边界扭曲, 容易过拟合; γ\gammaγ 小, 影响范围广, 决策边界平滑, 容易欠拟合
    • CCC: 惩罚系数, CCC 低使分界面平滑, CCC 高使分界面扭曲

损失函数: 合页损失 L(y,f(x))=max⁡(0,1−y⋅f(x))+λ∣∣w∣∣2L(y, f(x)) = \max(0, 1 - y \cdot f(x)) + \lambda ||w||^2L(y,f(x))=max(0,1−y⋅f(x))+λ∣∣w∣∣2, 如果分类正确且间隔足够大, 损失为 0, 否则损失随着错误成都线性增加

聚类

  • K-means: 基于欧氏距离, 假设簇是凸的
  • DBSCAN 密度聚类: 数据有复杂的几何形状, 如环形, S 型, 彼此嵌套. 密度聚类看样本点的连续密度, 按密度分离, 对噪声鲁棒性高
  • 高斯混合模型 GMM: 数据呈椭圆状/长条形分布, 或簇与簇间有明显重叠. GMM 在 K-means 基础上引入协方差, 允许簇在不同方向上有伸缩
  • 层次聚类: 最小距离层次聚类, 可以对同心圆作相关聚类. 对噪声和离群点敏感, 难以处理不同尺寸的簇和凸簇

  • bagging: 并行, 降低模型方差, 偏差不变

  • boosting: 串行, 降低模型偏差, 方差不变

  • Adaboost: 在弱学习期失败的样本上, 学习第二个弱学习器 (提升错误样本的权重)

    1. 初始化: 所有样本权重平均分配 w1,i=1Nw_{1,i} = \frac{1}{N}w1,i​=N1​
    2. 迭代训练
      1. 训练弱分类器: 基于当前样本权重分布 DmD_mDm​, 训练弱分类器 Gm(x)G_m(x)Gm​(x) (通常使用单层决策树)
      2. 计算误差率 ϵm=∑i=1Nwm,i⋅I(Gm(xi)≠yi)\epsilon_m = \sum_{i=1}^N w_{m,i} \cdot I(G_m(x_i) \neq y_i)ϵm​=∑i=1N​wm,i​⋅I(Gm​(xi​)​=yi​), 即错分样本的权重之和
      3. 计算分类器权重 αm=12ln⁡(1−ϵmϵm)\alpha_m = \frac{1}{2} \ln \left( \frac{1 - \epsilon_m}{\epsilon_m} \right)αm​=21​ln(ϵm​1−ϵm​​), ϵm\epsilon_mϵm​ 越小说明表现越好
      4. 更新样本权重 (提升分错样本的权重) wm+1,i=wm,iZmexp⁡(−αmyiGm(xi))w_{m+1,i} = \frac{w_{m,i}}{Z_m} \exp(-\alpha_m y_i G_m(x_i))wm+1,i​=Zm​wm,i​​exp(−αm​yi​Gm​(xi​))
    3. 构建强分类器: MMM 个弱分类器加权线性组合 f(x)=∑m=1MαmGm(x)f(x) = \sum_{m=1}^M \alpha_m G_m(x)f(x)=∑m=1M​αm​Gm​(x), 分类结果取符号 G(x)=sign(f(x))G(x) = \text{sign}(f(x))G(x)=sign(f(x))

    • 缺点: 性能取决于数据和弱学习器, 弱分类器过于复杂会过拟合, 过弱会欠拟合, 容易收到均匀噪声的影响

深度学习

激活函数

  • Sigmoid 为什么容易导致梯度消失?
    1. 导数最大值为 0.25: 反向传播时, 返回网络的误差将会在每一层收缩至少 75%
    2. 尾部饱和, 对应梯度接近 0, 导致梯度消失问题
  • Tanh: 关于原点对称, 手链更快
  • ReLU:
    1. 计算量小, 不涉及除法
    2. 一部分神经元输出为0: 网络系数, 减少了参数的相互依存关系, 缓解过拟合
    3. x > 0 时, 导数为 1, 解决了梯度消失问题, 收敛更快; x < 0 时, 导数为 0, 无法更新权重

神经网络

  • 逻辑回归: 结构是单个神经元 + sigmoid, 损失函数是交叉熵
  • 与核 SVM 的神经网络结构: 输入层 + 一个隐含层 + 阶跃函数 (激活函数), 损失函数是合页损失
