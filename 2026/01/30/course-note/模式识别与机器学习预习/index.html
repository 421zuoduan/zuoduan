计算题

最大似然估计

  1. 似然函数是出现当前所有数据的概率, 所以是所有数据概率的积
  2. 对似然函数求导, 导数等于 0, 求得极值点, 即参数

贝叶斯分类器

贝叶斯分类器的目标是最小化分类错误, 也即使后验概率 $P(w_i|x)$ 最大, 贝叶斯判别界面即分成两类概率相等的界面, $P(w_1|x) = P(w_2|x)$, 贝叶斯定理展开这个后验概率后整理得到贝叶斯判别界面 $P(x|y) = \frac{P(y|x)P(x)}{P(y)}$

对正态分布, 其最大似然结果为 $\hat{\mu} = \bar{x} = \frac{1}{N} \sum{i=1}^N x_i$, $\hat{\sigma}^2 = \frac{1}{N} \sum{i=1}^N (x_i - \hat{\mu})^2$

正态分布的判别函数为 $g_i(x) = -\frac{1}{2}(x-\mu_i)^T \Sigma^{-1} (x-\mu_i) + \ln P(\omega_i)$

Fisher 判别

寻找一个投影方向, 使得类内散度最小, 类间散度最大

  • 类内散度矩阵 $Sw = \sum_i\sum{x\in D_i}(x-\mu_i)(x-\mu_i)$, 衡量类内数据的离散程度, $\mu_i$ 是第 $i$ 类样本的均值向量
  • 类间散度矩阵 (以二分类为例) $S_b = (\mu_1-\mu_2)(\mu_1-\mu_2)^T$
  • Fisher 准则函数: 寻找向量 $w$, 使得瑞利商 $J(w)=\frac{w^TS_bw}{w^TS_ww}$ 最大, 分子是投影后的类间距离, 分母是投影后的类内方差之和
  • $J(w)$ 求导并令其为 0, 解得最优鉴别矢量 $w^$. 二分类的结果为 $w^ = S_w^{-1}(\mu_1-\mu_2)$
  1. 算两类样本的均值 $\mu_1$ 和 $\mu_2$
  2. 计算两类类内散度矩阵 $S1 = \sum{x\in D1}(x-\mu_1)(x-\mu_1)$, $S_2 = \sum{x\in D_2}(x-\mu_2)(x-\mu_2)$, 求得总类内散度 $S_w = S_1 + S_2$

  3. 套公式得 $w^{*T}x = z_0$

    其中 $w^* = S_w^{-1}(\mu_1-\mu_2)$, $z_0 = \frac{w^T(\mu_1+\mu_2)}{2}$

PCA 降维

核心目标: 找到数据方差最大的方向最作为主成分, 将数据投影到这些方向上从而实现去噪或压缩

  1. 计算所有样本的均值
  2. 计算中心化后的样本 $x_i = x_i - \overline{x}$, 一列一个样本组成 $X$
  3. 计算协方差矩阵 $S = XX^T$ (协方差矩阵并应该是 $\frac{1}{n-1}\sum (x_i-\bar{x})(x_i-\bar{x})^T$, 但是我们对样本已经做了去中心化, 而且仅关注关注方向, 所以去掉前面的缩放系数)
  4. 协方差矩阵特征值分解. 特征值代表数据在该方向上的分散程度, 特征向量就是投影的方向
  5. 投影: 选择最大的 k 个特征值对应的特征向量组成变换矩阵 $W$, 计算 $Z=W^TX$
  6. 重构: 将降至二维的样本重构会三维空间 $\hat{x}_i = Wz_i + \overline{x}$

选填

  • 朴素贝叶斯: $P(x_1, x_2, …, x_n | Y) \approx P(x_1|Y) \cdot P(x_2|Y) \cdot … \cdot P(x_n|Y)$, 假设各个特征间独立
  • 最小风险判别: 不使用最小错误率的策略, 而是使用最小风险的策略. 采取决策 $j$ 的风险为 $rj(x) = \sum{i=1}^{M} L{ij} P(\omega_i | x)$, 分类器判别 $x$ 属于 $w_j$ 类, 但实际来自 $w_i$ 类, 则分类器失败, 此时 $L{ij}$ 为失分, 对应的条件风险为后验概率进行
  • 判别式模型和生成式模型: 判别式模型学习条件概率分布, 只关心分界线在哪里; 生成式模型学习里阿尼和概率分布
  • 贝叶斯决策与贝叶斯判别
    • 贝叶斯决策理论中, 最小化错误率的决策规则是最大后验概率规则 MAP
    • 贝叶斯风险: 所有可能决策的期望损失, $R(\alpha|x) = \Sigma\lambda(\alpha|w_j)p(w_j|x)$, $\lambda$ 是损失函数
    • 贝叶斯决策理论中, 似然比检验 $\frac{p(x|\omega1)}{p(x|\omega_2)}$ 的阈值取决于先验概率和损失函数的比值 $\text{阈值 } \lambda = \underbrace{\frac{P(\omega_2)}{P(\omega_1)}}{\text{先验概率比}} \times \underbrace{\frac{\lambda{12} - \lambda{11}}{\lambda{21} - \lambda{22}}}_{\text{损失函数比}}$
    • 朴素贝叶斯分类器的 “朴素” 假设是: 特征间条件独立
  • 一对其余One-vs-Rest 和 一对一One-vs-One:
    • OvR: 多类情况1, 对于 K 个类别的分类问题, 训练 K 个二分类器, 最后输出概率最大的那个分类器对应的类别
    • OvO: 多类情况2, 训练 K(K-1)/2 个分类器, 投票输出选哪个
  • 半监督学习
    • 协同训练 Co-Training: 核心假设是多视图, 数据必须拥有两个充分且独立的特征子集 (特征互相独立, 且任何一个特征子集都能训练出来一个强分类器, 一个特征子集称为一个视图)
      1. 初始化: 基于有标签数据在两个视图上分别训练模型 A, B
      2. A 和 B 对无标签数据预测
      3. 筛选: 两个模型选出置信度最高的几个正例和负例
      4. 互教: 选出的样本加入对方的数据中, 重新训练
      5. 迭代: 扩充有标签数据, 直至用完所有数据
    • 标签传播算法
      • 核心思想: 基于图, 相似的数据点有相同的标签, 一个节点的标签取决于它的邻居们大多数是什么标签
      • 计算步骤
        1. 建图: 所有数据点视为图中节点, 节点间的边表示相似度 (使用 KNN 或 RBF 核函数建图)
        2. 传播-迭代: 节点根据邻节点标签的甲醛投票来更新自己的标签分布, “我属于 A 类的概率 = $\sum$ (邻居 $j$ 属于 A 类的概率 $\times$ 我和 $j$ 的连接权重)”
        3. 重置/钳制: 传播后, 有真实标签的数据重置回初始状态
        4. 收敛: 重复23, 直至所有节点标签无变化
    • EM 算法是半监督生成式模型 (如半监督朴素贝叶斯)
      • E: expectation 猜标签: 用当前模型预测无标签数据, 返回概率
      • M: Maximization 重新训练: 把猜出来的概率当作事实, 按照权重更新名参数
      • 迭代
  • 降维
    • LDA 线性判别分析: 监督学习, 最大化类间距离 (有标签才能这么做), 类内方差最小
    • PCA 主成分分析: 无监督学习, 找到数据方差最大的投影方向, 得到的各个主成分是正交的
    • t-SNE: 基于流行学习的非线性降维方法, 关注局部结构, 通过计算数据点间的概率分布来保持数据的局部邻域关系, 没有显式的投影函数, 不能直接应用于新数据. 可用来可视化
    • MDS 多维尺度分析: 将相似度或距离数据可视化为二维或三维坐标图, 保持高维空间中数据点间的相对距离
    • 流行学习的基本假设: 高维数据实际上分布在一个低维流形上
  • 集成学习
    • Adaboost: 分类器权重更新公式 $w_i^{(t+1)} = w_i^{(t)} \cdot \exp(-\alpha_t y_i G_t(x_i))$, 如果分错了：$y_i$ 和 $G_t(x_i)$ 符号相反，乘积为 -1。此时指数项变为 $\exp(-\alpha_t \cdot (-1)) = e^{\alpha_t}$
    • bagging 适合性能好, 深度深的决策树; boosting 适合性能弱, 深度浅的决策树
    • 超参调优: bagging 方法-包外估计(未被抽中来训练的数据); boosting 方法-交叉验证

知识点学习

有监督学习

L1 正则和 L2正则在损失函数后加一个惩罚项来限制模型参数 $w$ 的大小, 从而控制模型复杂度

  • L1 正则: $L = L0 + \lambda \sum{i} |w_i|$
  • L2 正则: $L = L0 + \lambda \sum{i} w_i^2$
特性 L1 正则化 (Lasso) L2 正则化 (Ridge)
主要效果 稀疏性 (Sparsity) 权重衰减 (Weight Decay)
参数变化 会将不重要的特征权重压缩为 0。 会将权重压缩得很小（趋近于 0），但几乎不会等于 0。
应用场景 特征选择：当你觉得特征很多，但只有少数几个有用时。 防止过拟合：处理共线性（Collinearity），让模型更平滑、稳定。
求解难度 在 0 点不可导（是一个尖角），需要特殊算法（如坐标下降法）。 处处可导，计算方便（闭式解或梯度下降均可）。
唯一解 在某些情况下可能有多个解。 总是能得到唯一解（增加了目标函数的凸性）。
  • L1 的图形是“菱形” (Diamond Shape):$|w_1| + |w_2| \le C$ 的图形是一个有尖角的菱形。损失函数的等高线向中心收缩时，很容易先碰到菱形的顶点（角上）。在顶点处，往往意味着某个坐标轴的值为 0（例如 $w_1=0, w_2=C$）。这就是为什么 L1 会产生稀疏解（让某些权重变 0）。
    • L2 的图形是“圆形” (Circle Shape):$w_1^2 + w_2^2 \le C$ 的图形是一个圆。损失函数的等高线更容易碰到圆周上的任意一点，而不是刚好在坐标轴上。所以 $w_1$ 和 $w_2$ 都会被压缩变小，但很难同时出现某一个变为 0 的情况。

支持向量机

支持向量: 离决策边界最近的样本点, SVM 的决策边界只由支持向量决定, 与其他样本点无关
间隔 Margin: 支持向量到超平面的距离的垂直距离

  • 硬间隔: 现行可分

  • 软间隔: 允许少来那个样本错误或位于间隔内

    • 松弛变量 $\xi$: 为每个样本引入一个变量, 量化它未被间隔的程度
    • 惩罚系数 $C$: C 很大, 对错误零容忍, 容易过拟合; C 很小, 追求更宽的间隔, 容易欠拟合

核函数

核函数: 处理非线性. 低维空间无法线性分割时, 使用核函数将数据映射到高维特征空间

常见核函数:

  1. 线性核 $K(x, y) = x^T y$: 适用于特征维度很高, 或数据本身线性可分的情况
  2. 多项式核: 引入多项式特征
  3. 高斯核 RBF Kernel $K(x, y) = \exp(-\gamma ||x-y||^2)$: 最常用, 可以将数据映射到无穷维空间
    • 参数 $gamma$: 控制高斯分布的宽度. $\gamma$ 大, 附近的点影响大, 决策边界扭曲, 容易过拟合; $\gamma$ 小, 影响范围广, 决策边界平滑, 容易欠拟合
    • $C$: 惩罚系数, $C$ 低使分界面平滑, $C$ 高使分界面扭曲

损失函数: 合页损失 $L(y, f(x)) = \max(0, 1 - y \cdot f(x)) + \lambda ||w||^2$, 如果分类正确且间隔足够大, 损失为 0, 否则损失随着错误成都线性增加

聚类

  • K-means: 基于欧氏距离, 假设簇是凸的
  • DBSCAN 密度聚类: 数据有复杂的几何形状, 如环形, S 型, 彼此嵌套. 密度聚类看样本点的连续密度, 按密度分离, 对噪声鲁棒性高
  • 高斯混合模型 GMM: 数据呈椭圆状/长条形分布, 或簇与簇间有明显重叠. GMM 在 K-means 基础上引入协方差, 允许簇在不同方向上有伸缩
  • 层次聚类: 最小距离层次聚类, 可以对同心圆作相关聚类. 对噪声和离群点敏感, 难以处理不同尺寸的簇和凸簇

半监督学习

  • 平滑假设: 高密度区域中 $x_1$ 和 $x_2$ 接近, 对应输出 $y_1$, $y_2$ 也接近
  • 聚类假设: 两个点在同一个簇, 很有可能属于一类. 等价于低密度分割 (决策边界在低密度区域)
  • 流形假设: 输入空间由所有数据点所在的多个低维流形构成, 位于同一流形上的数据点具有相同的标签, 流形上距离近的点的标签相似

集成学习

  • bagging: 并行, 降低模型方差, 偏差不变
  • boosting: 串行, 降低模型偏差, 方差不变

  • Adaboost: 在弱学习期失败的样本上, 学习第二个弱学习器 (提升错误样本的权重)

    1. 初始化: 所有样本权重平均分配 $w_{1,i} = \frac{1}{N}$
    2. 迭代训练
      1. 训练弱分类器: 基于当前样本权重分布 $D_m$, 训练弱分类器 $G_m(x)$ (通常使用单层决策树)
      2. 计算误差率 $\epsilonm = \sum{i=1}^N w_{m,i} \cdot I(G_m(x_i) \neq y_i)$, 即错分样本的权重之和
      3. 计算分类器权重 $\alpha_m = \frac{1}{2} \ln \left( \frac{1 - \epsilon_m}{\epsilon_m} \right)$, $\epsilon_m$ 越小说明表现越好
      4. 更新样本权重 (提升分错样本的权重) $w{m+1,i} = \frac{w{m,i}}{Z_m} \exp(-\alpha_m y_i G_m(x_i))$
    3. 构建强分类器: $M$ 个弱分类器加权线性组合 $f(x) = \sum_{m=1}^M \alpha_m G_m(x)$, 分类结果取符号 $G(x) = \text{sign}(f(x))$
    • 优点: 实现快速简单, 灵活, 通用性高
    • 缺点: 性能取决于数据和弱学习器, 弱分类器过于复杂会过拟合, 过弱会欠拟合, 容易收到均匀噪声的影响
    • 为什么 AdaBoost 在训练误差为 0 后继续训练可能造成测试误差的持续下降: 更多的训练会增加分类器的分类margin, 这个过程会防止测试误差的上升, 随着 margin的变大, 测试误差会逐渐收敛

深度学习

激活函数

  • Sigmoid 为什么容易导致梯度消失?
    1. 导数最大值为 0.25: 反向传播时, 返回网络的误差将会在每一层收缩至少 75%
    2. 尾部饱和, 对应梯度接近 0, 导致梯度消失问题
  • Tanh: 关于原点对称, 手链更快
  • ReLU:
    1. 计算量小, 不涉及除法
    2. 一部分神经元输出为0: 网络系数, 减少了参数的相互依存关系, 缓解过拟合
    3. x > 0 时, 导数为 1, 解决了梯度消失问题, 收敛更快; x < 0 时, 导数为 0, 无法更新权重

神经网络

  • 逻辑回归: 结构是单个神经元 + sigmoid, 损失函数是交叉熵
  • 与核 SVM 的神经网络结构: 输入层 + 一个隐含层 + 阶跃函数 (激活函数), 损失函数是合页损失
