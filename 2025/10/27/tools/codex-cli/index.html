配置端口转发

首先确认端口是否被占用

  1. 在本地 codex 插件登录 openai 账号, 然后将本地 user/.codex/ 文件夹移动到服务器 /home/user/.codex
  2. 配置本地端口, 我修改了 user/.ssh/config 文件, 注意下面 15554 是本地代理的端口, 比如用 clash 要看 clash 使用的端口
  3. 配置服务器端口, 修改 .vscode-server/data/Machine/settings.json, 也即修改远程服务器的 setting.json 文件, 下面 15555 是服务器转发端口, 配置如下
  4. 测试端口转发是否成功, 返回 ip 和无效的 api key 就算转发成功
  5. 配置 ssh 连接自启动服务, .bashrc 加入以下指令
  6. 刷新 ssh 连接列表, 重启 vscode 或 ssh 连接
