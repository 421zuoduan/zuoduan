线性方程组Ⅰ

高斯消元法

逐列作为主元, 消去主元下面的元素, 直至矩阵变成三角阵

高斯-约旦法

高斯消元法基础上, 保证主元系数为 1, 同时每一列最多只有一个主元

部分主元法

高斯消消去法基础上, 选取主元时优先选取系数绝对值最大的作为主元, 以保证消去其他行时其他行乘上的系数绝对值小于 1

完全主元法

每一步的主元都选择该位置往右下全部值的最大值 (没确定主元部分的最大值), 为选择最大值可以同时交换行和列, 但要记得怎么交换的

线性方程组Ⅱ

修正高斯消元法

高斯消元法基础上, 每一行主元都选择非 0 系数, 最后得到阶梯状矩阵,

矩阵代数

A*

共轭转置: 转置的同时把虚数符号取反. 定义为 $A=\overline{A}^T=\overline{A^T}$, 有 $(A)*=A$

$(A+B)^T=A^T + B^T$, $(A+B) = A+B$, $(\alpha A) = \overline{\alpha}A*$

线性性

证明 $f(\alpha x + b) = \alpha f(x) + f(b)$

矩阵性质

$AA^T$ 和 $A^TA$ 都为对称矩阵

$\mathrm{trace}(AB) = \mathrm{trace}(BA)$

奇异矩阵/可逆矩阵

奇异矩阵是不可逆矩阵, $rank(A)<n$ 时矩阵不可逆, 行列式为 0, 矩阵奇异

可逆矩阵相乘结果仍是可逆矩阵

矩阵乘以非奇异矩阵不会改变矩阵的秩

当且仅当 $rank(A)=rank(B)$ 时, $A~B$, 即 A 与 B 等价

$rank(A) = rankA^T, rank(A)=rank(A)$

谢尔曼莫里森公式

诺依曼级数

当 $\lim {n\rightarrow\infty}A^n = 0$ 时, $I-A$ 是非奇异的且 $(I-A)^{-1} = I + A + A^2 + \cdots = \sum{k=0}^\infty A^k$

LU 分解

非奇异矩阵 A 可进行 LU 分解的充要条件是: 在使用第三类初等行变换将其化为上三角矩阵过程中, 不出现零主元

在不允许行交换时: U 是高斯消元法得到的上三角矩阵, L 是行变换代表的矩阵对角线下方值取反得到的矩阵. 求解 $Ax=b$ 时先求解 $L(Ux)=b$ 的 $Ux$, 然后再求解 $x$

允许行交换时: $PA=LU$, 通过 [A|p] 得到 $P, L, U$, 求解 $Ax=b$ 时先求解 $PAx=L(Ux)=Pb$ 的 $Ux$, 然后再求解 $x$

上述 LU 分解得到的 U 对角线不是 1, 可以通过将每行除以对角线元素值来将对角线化为 1, 额外的一个对角矩阵 $D$ 对角线即原 $U$ 对角线元素. 即 $A=LDU$, 该分解具有唯一确定性

向量空间

向量空间的提出是为了统一数学实体. 一个向量空间涉及四个要素: $\mathcal{V}$, $\mathcal{F}$, 向量加法和标量乘法. $\mathcal{V}$ 是 $\mathcal{F}$ 上的向量空间, $\mathrm{F}$ 一般指实数域 $\mathcal{R}$ 或复数域 $\mathcal{C}$

向量空间要满足加法封闭性 $x, y\in \mathcal \Rightarrow x+y\in \mathcal{S}$ 和数乘封闭性 $x\in \mathcal{S}\Rightarrow \alpha x\in \mathcal{S}$, 零向量必定在向量空间中

子空间可以理解为过原点的平坦表面

向量空间子集向量集合 $\mathcal{S} = {v_1, v_2, \dots, v_r }$的可能线性组合为 $span(\mathcal{S})={\alpha_1 v_1 + \alpha_2 v_2 + \cdots + \alpha_rv_r}$, $span(\mathcal{S})$ 是 $\mathcal{V}$ 的子空间

总结

$T$ 是操作, $L$ 是第二个操作, $P$ 是从基 $\mathcal{B}$ 到 $\mathcal{B}’$ 的基变换矩阵, $([A]_{\mathcal{B}})$ 是线性算子 $A$ 在基 $(\mathcal{B})$ 下的坐标矩阵

  1. $P = [T]{\mathcal{B}} = [T]{\mathcal{B}’} = [I]{\mathcal{BB}’} = \big([x_1]{\mathcal{B}’}\big|[x2]{\mathcal{B}’}\big|\cdots [xn]{\mathcal{B}’}\big)$, 是从基 $\mathcal{B}$ 到 $\mathcal{B}’$ 的基变换矩阵, 也是非奇异矩阵
  2. $T(new basis) = old basis$
  3. 坐标操作: $new coordinates = P(old coordinates)$, 向量的坐标操作 $[v]{\mathcal{B}’} = P[v]{\mathcal{B}}$
  4. $[A]{\mathcal{B}} = P^{-1}[A]{\mathcal{B}’}P, \mathrm{where} P=[I]_{\mathcal{BB}’}$
  5. $[T^{-1}]{\mathcal{B}} = [T]{\mathcal{B}}^{-1} = P^{-1}$

模与内积Ⅰ

复数域中, $A^*$ 相当于对虚数部分取反, 然后转置, 没有虚数部分时就相当于是转置矩阵

CBS 不等式, 对于 $x, y\in \mathcal{C}^{n\times1}$, 当且仅当 $y=\alpha x, \alpha = \frac{x^y}{x^x}$ 时取等. CBS 不等式的重要性在于将内积和范数联系起来

1范数, 2范数, 无穷范数

F范数 $|A|F^2 = \sum{i,j}|a{ij}|^2 = \sum_i|A{i}| = trace)A^A$, 有 $|A+B|\leq |A| + |B|$, $|AB|\leq|A||B|$

诱导矩阵范数: 由向量范数得到矩阵范数 $|A| = \max_{|x|=1}|Ax|$, 其中 $A\in\mathcal{C}^{m\times n}, x\in \mathcal{C}^{n\times 1}$

由二范数诱导的矩阵范数为 $|A|2 = \max{|x|2=1}|Ax|_2=\sqrt{\lambda{\max}}$, $\lambda_{\max}$ 是 $A^*A$ 的最大特征值

由一范数诱导的矩阵范数为 $|A|1 = \max{|x|1}|Ax|_1 = \max_j\sum_i|a{ij}| = 绝对值和最大的一列$

由无穷范数诱导的矩阵范数为 $|A| = \max{|x|{\infty}=1}|Ax|\infty = \max_i\sum_j|a{ij}| = 绝对值和最大的一行$

格拉姆-施密特正交化方法

Gram-Schmidt 正交化: $n$ 维内积空间 $\mathcal{S}$ 有基 $\mathcal{B} = {x_1, x_2, \cdots, x_n}$, 用 $\mathcal{B}$ 构造标准正交基 $\mathcal{O} = {u_1, u_2, \cdots, u_k }$

  1. 取 $k=1$, $\mathcal{O} = {u_1}$, $u_1 = \frac{x_1}{|x_1|}$
  2. $\mathcal{B}$ 是内积空间 $\mathcal{S}$ 的基时, $ $u = \frac{xk-\sum{i=1}^{k-1}ui}{|x_k-\sum{i=1}^{k-1}ui|}$; 若 $\mathcal{S}$ 是 $\mathcal{C}^{m\times 1}$ 的 $n$ 维子空间, $u_k = \frac{(I-U_kU_k^T)x_k}{|(I-U_kU_k^T)x_k|}$, 这里 $U_1=0{m\times 1}$

QR 分解

列独立的矩阵 $A{m\times n} = Q{m\times n}R_{n\times n}$, 其中 $Q$ 是$R(A)$ 的标准正交基矩阵, $R$ 是对角线元素为正的上三角矩阵. LU 分解 和 QR 分解的结果由 $A$ 唯一确定, LU 分解要求矩阵是方阵, QR 分解只要求列向量线性无关

推导上, 对 $A$ 进行 Gram-Schmidt 正交化, 得到的就是 $Q$, $R$ 为以下形式, 其中 $v1 = |a_1|$, $v_k = |a_k-\sum{i=1}^{k-1}q_i|$

这里 $Q$ 的列向量是标准正交基, 因此 $Q$ 是正交矩阵, 正交矩阵有 $Q^T = Q^{-1}$ (因为 $Q^TQ=I$).

如果 $rank(A_{m\times n}) = n$ (列满秩, 即列向量线性无关), $A=QR$, 那么 $Rx = Q^Tb$ 的解是 $Ax=b$ 的解/最小二乘解. 这个形式与 LU 分解类似

Modified Gram-Schmidt

经典 Gram-Schmidt 正交化在浮点数运算中会产生非正交的结果, 可以通过引入投影矩阵 $Ei=I-u{i-1}u{i-1}^*$ 消除向量在 $u{i-1}$ 上的投影分量. 这样调整了投影操作的顺序: 没得到一个新的正交向量 $u_k$, 用它去正交化后面所有未处理的向量, 从而减少误差

  1. $u_1 = \frac{x_1}{|x_1|}$
  2. $uj\leftarrow u_j - (u{k-1}^*yj)u{k-1}$, 对更新后的 $uk$ 归一化 $u_k = \frac{u{k}}{|u_k|}$

酉矩阵和正交矩阵

酉矩阵就是复数域里的正交矩阵, 正交矩阵是酉矩阵的一个子集. 正交矩阵的行向量和列向量都是正交的

  1. $U^U=I\Leftrightarrow U^{-1}=U^\Leftrightarrow UU^*=I$
  2. 矩阵乘以正交矩阵不会改变长度, 只会改变方向 (平方验证)
  3. 单位投影矩阵 $Q = I-uu^$, 其中 $u\in\mathcal{C}^{n\times 1}$ 且 $|u|=1$, 几何意义是把 $x$ 投影到了 $u$ 的正交补空间. 同理 $uu^x$ 是把 $x$ 映射到 $span{u}$ 上, 也即投影到 $u$ 在的这条直线 (一个简单的理解方法 $uu^x=u(u^x)$, 结果是在 $u$ 所在的直线上)
  4. 正交投影算子: 能将任意向量投影到目标子空间. $P_u x = \frac{Puu^}{u^u} x$ 是将 $x$ 映射到 $u$ 所在的直线上

初等反射变换 (Householder 变换)

关于 $u\in \mathcal{C}^{n\times 1}$ 的正交补空间的初等反射变换为 $R=I-2\frac{uu^}{u^u}$. $Rx$ 是关于 $u$ 的正交补空间的镜像反射结果, 有

这里 $Q=P_{u^{\perp} = I-\frac{uu^}{u^u}}$ 是 $x$ 到 $u^{\perp}$ 的正交投影算子.

$I-\frac{uu^}{u^u}$ 是把 $x$ 投影到 $u^{\perp}$ 上, 再减 $\frac{uu^}{u^u}$ 能朝镜面另一方向翻折

Householder 变换

将向量反射到第一坐标轴

  1. 确定向量 $x$ 并计算范数 $|x|$
  2. 构造 $u$: 对第一个元素非零的向量 $x$ 构造 $u = x\plusmn \mu|x|e_1$, $e_1=(1, 0, \dots, 0)^T$ 是第一个标准基向量, $\mu$ 是 $x_1$ 方向的单位标量, $x_1$ 为实数时为 1, 否则为 $\frac{x_1}{|x_1|}$
  3. 将计算 Householder 变换矩阵 $R$: $u$ 构造的 Householder 变换作用于 $x$ 有 $Rx = \mp\mu|x|e_1$, 结果只有第一个分量非零, 即 $x$ 被反射到第一坐标轴上
  4. 验证 $Rx$ 是否在坐标轴上

$u$ 的几何意义是反射镜面的饭向量, $u^\perp$ 是反射镜面, $R$ 的几何意义是关于镜面 $u^\perp$ 的镜像反射变换

正交化简

Householder reduction: 反射矩阵 $R_k$ 用于矩阵化简

  1. $A{m\times n} = [A{1}|\cdots|A_{n}]$
  2. 构造 $u=A{*1}\pm \mu|A{*1}|e_1$
  3. 用 $A$ 的第一列 $x=A_{1}$ 构建反射矩阵 $R_1 = I-2\frac{uu^}{u^*u}$
  4. 得到映射方程 $R1A{1} = \mp\mu|A_{1}|e1 = (t{11}, 0, \cdots, 0)^T$
  5. 得到 $R1A = [R_1A{1}|R2A{2}|\cdots|RnA{*n}] = \begin{pmatrix}t_{11}& t_1^T\ 0&A_2\end{pmatrix}$, 这里 $A_2$ 形状是 $m-1\times n-1$
  6. 对上面结果右下角 $A_2$ 应用相同步骤, 能得到形状为 $m-1\times n-1$ 的 $\hat{R}_2$, 为了不改变已化简的行, 用 $\hat{R}_2$ 组成 $R_2 = \begin{pmatrix}1&0\ 0& \hat{R}_2 \end{pmatrix}$
  7. 得到 $R_2R_1A = \begin{pmatrix}t_11 & t_1^T\ 0& \hat{R}_2A_2\end{pmatrix}$, 依次完成所有列的计算
  8. $k-1$ 步后, 得到 $R{k-1}\cdots R_2R_1A = \begin{pmatrix}T{k-1}& \tilde{T}_{k-1}\ 0& A_k \end{pmatrix}$

也即: 对任意 $A\in \mathcal{C}^{m\times n}$, 存在酉矩阵 $P$ 使得 $PA=T$, 其中 $T$ 是上梯形矩阵

正交化简的最小二乘法

正交归约法可用于求解不相容方程 $Ax=b, m\geq n$

  1. 目标: 找 $x$, 使误差 $\omega = Ax-b$ 平方最小
  2. 对 $A$ 做正交约化, 得到上提醒矩阵 $T$, 分块为 $T=\begin{pmatrix}R_{n\times n\0} \end{pmatrix}$
  3. 对 $b$ 做同样的正交变换, 得到 $Pb = \begin{pmatrix}c_{n\times 1}\d \end{pmatrix}$
  4. $PAx = Pb$, $|\omega|^2 = |P\omega|^2 = |PAx-Pb|^2 = |Rx-c|^2+|d|^2$, 即 $x$ 是上三角矩阵的 $Rx=c$ 的解

有四种将矩阵化为上三角阵的不同方法

  1. 高斯消元法
  2. Gram-Schmidt 正交化
  3. Householder 变换
  4. Givens 旋转变换

其中 Householder 和 Givens 变换是浮点数下的稳定算法, 用于生成 $n\times n$ 的 QR 分解

互补子空间

向量空间 $\mathcal{V}$ 的两个子空间 $\mathcal{X}$, $\mathcal{Y}$ 满足以下条件时为互补子空间, $\mathcal{V}$ 是 $\mathcal{X}$ 和 $\mathcal{Y}$ 的直和, 记为 $\mathcal{V}=\mathcal{X}\oplus\mathcal{Y}$

  1. $\mathcal{V}=\mathcal{X}+\mathcal{Y}$
  2. $\mathcal{X}\cap\mathcal{Y}{0}$

直和的等价条件:

  1. $\mathcal{V}=\mathcal{X}\oplus\mathcal{Y}$
  2. $\mathcal{V}$ 中的任意向量 $v$, 都能唯一分解为 $v=x+y$
  3. $\mathcal{B}{\mathcal{X}}$ 与 $\mathcal{B}{\mathcal{Y}}$ 无交集, 且并集是 $\mathcal{V}$ 的基

$x$ 是 $v$ 沿着 $\mathcal{Y}$ 到 $\mathcal{X}$ 的投影, 定义线性算子 $P$ 为沿着 $\mathcal{Y}$ 到 $\mathcal{X}$ 的投影算子, $Pv=x$

投影算子的性质:

  1. 幂等性: $P^2=P$
  2. 互补投影算子: $I-P$ 是沿着 $\mathcal{X}$ 到 $\mathcal{Y}$ 的投影
  3. 像空间是不动点集: $R(P) = {Px=x}$
  4. 像空间与零空间对应 $R(P) = N(I-P) = \mathcal{X}$, $R(I-P)=N(P)=\mathcal{Y}$
  5. 互补子空间的基矩阵分别为 $X, Y$, $[X|Y]$ 是可逆矩阵
  6. 计算沿 $\mathcal{Y}$ 在 $\mathcal{X}$ 上的投影, 关键是 $P[X|Y] = [X|0]$

值域-零空间分解 range-Nullspace

奇异矩阵 $A$, $\mathcal{R}^n = R(A^k)\oplus N(A^k)$, 此时最小的 $k$ 是 $A$ 的指数. 对于非奇异矩阵, $index(A)=0$

  1. $rank(A^k)=rank(A^{k+1})$
  2. $R(A^k) = R(A^{k+1})$
  3. $N(A^k) = N(A^{k+1})$

幂零分解 core-nilpoten

幂零矩阵: $N_{n\times n}$ 满足 $N^k=0$

只能用于奇异矩阵, 核心是将奇异矩阵通过相似变换, 拆成非奇异核与幂零部分组成的块对角矩阵. $index(A)=k$, $rank(A^k)=r$

存在一个非奇异矩阵 $Q$, 使得 $A$ 通过相似变换化为块对角矩阵 $Q^{-1}AQ=\begin{pmatrix}C{r\times r}&0\ 0&N\end{pmatrix}$, 其中 $C{r\times r}$ 是非奇异矩阵, $N$ 是幂零矩阵

Drazin 逆是奇异矩阵的一种广义逆 (狭义逆只有非奇异矩阵有) $A^D = Q\begin{pmatrix}C^{-1}&0\0&0 \end{pmatrix}Q^{-1}$

正交分解

内积空间 $\mathcal{V}$, $\mathcal{M}\subseteq$, $\mathcal{M}$ 的正交补 $\mathcal{M}^{\perp} = {x\in\mathcal{V}|=0\text{ for }m\in\mathcal{M}}$, 正交补与 $mathcal{M}$ 中的每个向量都正交

$\mathcal{V}=\mathcal{M}\oplus\mathcal{M}^{\perp}$, 对所有矩阵都可用

URV因式分解

秩为 $r$ 的矩阵 $A_{m\times n}$, 存在 $m\times m$ 的正交矩阵 $U$, $n\times n$ 的正交矩阵 $V$, $r\times r$ 的非奇异矩阵 $C$, 有:

$A=URV^T=U\begin{pmatrix}C{r\times r}&0\ 0&0\end{pmatrix}{m\times n}V^T$

这里 $U$ 和 $V$ 的列是四个基本子空间的标准正交基

  1. $U$ 的前 $r$ 列是 $R(A)$ 的标准正交基
  2. $U$ 的后 $m-r$ 列: $N(A^T)$ 的标准正交基
  3. $V$ 的前 $r$ 列是 $R(A^T)$ 的标准正交基
  4. $V$ 的后 $m-r$ 列: $N(A)$ 的标准正交基

由于标准正交基的选取不同, URV 分解的结果并不唯一

非奇异矩阵是平凡的 RPN 矩阵, 有以下性质

  1. $R(A)\perp N(A)$
  2. $R(A)=R(A^T)$
  3. $N(A)=N(A^T)$
  4. $A$ 可分解为 $A=URU^T=U\begin{pmatrix}C_{r\times r}&0\ 0&0\end{pmatrix}U^T$

正交投影

讨论实数域内互补子空间正交的情况. 对于 $\mathcal{N}^n$的一对互补子空间 $\mathcal{M}$, $\mathcal{N}$, 沿 $\mathcal{N}$ 到 $\mathcal{M}$ 的投影算子 $P=(M|N)\begin{pmatrix}I&0\ 0&0\end{pmatrix}(M|N)^{-1}=(M|0)(M|N)^{-1}$

  1. $P{\mathcal{M}} = M(M^TM)^{-1}M^T$, $P{\mathcal{M^{\perp}}} = N(N^TN)^{-1}N^T$
  2. $P_{\mathcal{M}}=U\begin{pmatrix}I_r&0\0&0\end{pmatrix}U^T$, 这里 $U=(M|N)$
  3. $P{\mathcal{M}^{\perp}} = I-P{\mathcal{M}}$

矩阵行列式

一个排列 $p$ 可通过偶数(奇数)次兑换回复为自然顺序, 那其他能恢复成自然顺序的方法也必定是偶数(奇数)次. 行列式必须是方阵

三角阵的行历史是对角线元素的乘积

  1. $\det(A^T) = \det (A)$
  2. 交换行: $\det(B) = -\det(A)$
  3. 某行乘 $\alpha$: $\det(B) = \alpha\det(A)$
  4. 某行乘 $\alpha$ 加到另一行: $\det(B) = \det(A)$
  5. $\det(AB) = \det(A)\det(B)$
  6. 当且仅当 $\det(A)\neq 0$ 时 $A$ 是非奇异矩阵, 且较小的行列式值并非更接近非奇异性质
  7. $rank(A)=A中最大非零子式的阶数$
  8. $\det\begin{pmatrix}A&B\0&D\end{pmatrix} = \det(A)\det(D)$, 要求 $A, D$ 均为方阵
  9. $\det\begin{pmatrix}A&B\ C&D\end{pmatrix} = \begin{cases}\det(A)-\det(D-CA^{-1}B)\qquad \mathrm{when}\quad A^{-1} \quad \mathrm{exist}\ \det(D)\det(A-BD^{-1}C)\qquad \mathrm{when}\quad D^{-1}\quad\mathrm{exist}\end{cases}$, 也即 $\begin{pmatrix}A&B\ C&D\end{pmatrix} = \begin{pmatrix}I&0\ CA^{-1}&I\end{pmatrix}\begin{pmatrix}A&B\ 0& D-CA^{-1}B\end{pmatrix}$
  10. $\det(I+cd^T) = 1 + d^Tc$, $\det(A+cd^T) = \det(A)(1+d^TA^{-1}c)$
  11. 克拉默法则: 对于 $A_{n\times n}x=b$, 方程组第 $i$ 个未知数 $x_i = \frac{\det(A_i)}{\det(A)}$, 其中 $A_i$ 是将第 $i$ 列替换为常数向量 $b$ 后的矩阵

特征值和特征向量

$A$ 的不同特征值的集合记为 $\sigma(A)$, 也称为 $A$ 的谱

特征多项式: $p(\lambda) = \det(A-\lambda I) = \lambda^n + c_1\lambda^{n-1}+c_2\lambda^{n-2}+\cdots+c_n=0$, 系数 $c_k = (-1^k)\cdot\sum(所有k\times k 主子式)$

特征方程: $p(\lambda) = \det(A-\lambda I) = 0$, 特征方程的解即所有特征值

$r$ 阶主子矩阵: 选中 $A$ 中下标相同的 $r$ 个行和 $r$ 个列, 这些行列组成的 $r\times r子矩阵$. 主子式是主子矩阵的行列式

  1. $\det(A) = \lambda_1\lambda_2\cdots\lambda_n = (-1)^nc_n$
  2. $trace(A) = \lambda_1+\lambda_2+\cdots+\lambda_n = -c_1$
  3. 谱半径 $\rho(A) = \max_{\lambda\in\sigma(A)}|\lambda|$
  4. $A$ 与 $B$ 相似: $P^{-1}AP=B$
  5. 可对角化: $A$ 相似于一个对角矩阵, 即 $P^{-1}AP = \Lambda$时 $A$ 可对角化, $\Lambda$ 是对角矩阵. 充要条件: 当且仅当 $A$ 有 $n$ 个线性无关的特征向量. $P$ 的列向量与特征值构成特征对, 也就是一个特征向量.
  6. 不是所有方阵都能对角化, 但是所有方阵都能相似变换化为三角矩阵. 这个相似变换是酉变换
  7. 舒尔三角化定理: 任意方阵 $A$ 存在一个酉矩阵 $U$ 使得 $U^*AU=T$, $T$ 是上三角矩阵, 对角线为$A$ 的所有特征值
