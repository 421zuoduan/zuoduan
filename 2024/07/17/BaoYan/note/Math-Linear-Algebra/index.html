行列式

  1. ∣A∣=∣AT∣|A| = |A^T|∣A∣=∣AT∣

  2. 行列式两行或两列互换位置, 行列式的值变号

  3. 两行或两列相同, 行列式的值为 0

  4. 某行 (或列) 如有公因子 k, 可将 k 提出行列式记号外 (数 k 乘行列式 ∣A∣|A|∣A∣ 等于用 k 乘它的某行 (或列))

  5. 行列式某行 (或列) 是两个元素之和, 可将行列式拆为两个行列式之和

    a_1+b_1& a_2+b_2& a_3+b_3\
    c_1& c_2& c_3\
    d_1& d_2& d_3
    \end{vmatrix} =
    \begin{vmatrix}
    a_1& a_2& a_3\
    c_1& c_2& c_3\
    d_1& d_2& d_3
    \end{vmatrix} +
    \begin{vmatrix}
    b_1& b_2& b_3\
    c_1& c_2& c_3\
    d_1& d_2& d_3
    \end{vmatrix}

n 阶矩阵可逆的充分必要条件

  1. 存在 n 阶矩阵 BBB, AB=EAB = EAB=E
  2. ∣A∣≠0|A|\neq 0∣A∣=0, 或 r(A)=nr(A) = nr(A)=n, 或 AAA 的列 (行) 向量线性无关
  3. 齐次方程组 Ax=0Ax=0Ax=0 只有零解
  4. 任意 bbb, 非齐次线性方程组 Ax=bAx=bAx=b 总有唯一解
  5. 矩阵 AAA 的特征值不全为 0
