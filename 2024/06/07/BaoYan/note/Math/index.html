对偶

弱对偶定理: 如果原问题和对偶问题都存在最优解, 则对偶问题的最优值不大于原问题的最优值

  • 如果原问题对最优解的函数值与对偶问题的最优解的函数值相等, 称为强对偶
  • 如果原问题对最优解的函数值小于对偶问题的最优解的函数值, 称为弱对偶

机器学习

实际机器学习中为什么经常假设样本数据服从正态分布?

  1. 中心极限定理保证, 多个独立同分布的随机变量之和服从正态分布
  2. 正态分布的各种积分, 包括数学期望, 方差, 协方差, 边缘分布, 条件分布都易于计算, 可以得到解析解
  3. 给定数学期望和方差, 正态分布的熵最大

基本概念

有监督学习, 无监督学习

有监督学习: 有训练过程, 算法用训练集进行学习, 用学习得到的模型进行预测, 样本由输入值和标签组成. 有监督学习的目标是给定训练样本集, 确定假设空间中的映射函数

无监督学习: 对无标签的样本进行分析, 发现样本集的结构或者分布规律, 例如聚类, 表示学习和数据降维.

聚类也是分类问题, 但没有训练过程, 算法将一批没有标签的样本划分为多个类, 使每一类中的样本尽量相似, 不同类的样本间尽量不同

分类问题和回归任务

有监督学习中, 如果样本标签是整数则称为分类任务. 此时样本的类别以整数编号, 预测函数是向量向整数的映射

Rn→Z\mathbb{R}^{n}\rightarrow \mathbb{Z} Rn→Z

如果标签值是连续实数则称为回归问题, 此时预测函数是向量到实数的映射

Rn→R\mathbb{R}^n\rightarrow \mathbb{R} Rn→R

常见的有监督学习算法和无监督学习算法

有监督学习算法: 贝叶斯分类器, 决策树, 支持向量机, boosting 算法

无监督学习算法: K 均值算法, PCA, t-SNE, 层次聚类

强化学习

**原理: **模拟人的行为, 通过对状态-动作-回报值序列进行学习, 得到一个称为策略函数的模型, 用于确定每种状态下要执行的动作. 训练时对正确的动作奖励, 惩罚错误的动作, 用训练完成后的模型进行预测

生成模型和判别模型

假设 x\textrm{x}x 为特征向量, yyy 为该样本的标签值

如果对样本特征向量和标签的联合概率分布 p(x,y)p(\mathrm{x,y})p(x,y) 建模, 称为生成模型.

如果对条件概率 p(x,y)p(\mathrm{x}, y)p(x,y) 建模, 称为判别模型.

不适用概率模型的分类算法也属于判别模型, 它直接预测样本的标签值而不关心样本的概率分布, 这种情况的预测函数是

y=f(x)y = f(\textrm{x}) y=f(x)

这三种模型分别称为生成学习, 条件学习和判别学习

概率模型一定是生成模型吗? 不一定, logistic 回归和 softmax 回归都是概率模型, 但不是生成模型

生成模型: 贝叶斯分类器, 隐马尔可夫模型, 受限玻尔兹曼机

判别模型: 决策树, 全神经网络, 支持向量机, logistic 回归, AdaBoost 算法, 条件随机长

过拟合和欠拟合

过拟合

如何判断过拟合?

模型在训练集上精度高, 测试集上精度低

如何缓解过拟合?

  1. 正则化

正则化用于控制模型复杂度, 防止模型在训练数据上过拟合. 在模型的损失函数中加入额外的惩罚项, 用以对模型参数进行约束, 降低模型复杂度

正则化的作用:

  1. 防止过拟合
  2. 提高模型的泛化能力
  3. 减少模型的复杂度
  4. 控制模型学习速度
  5. 提高模型的鲁棒性

正则化的具体方法:

  1. L1 正则化. 添加 L1 范数, L1正则化倾向于产生稀疏权重矩阵, 将一些权重置零, 可以实现特征选择的效果
为什么 L1 正则化可以实现特征选择? 对于 $Ax=b$, $A$ 是训练数据, $x$ 是解向量, $b$ 是标签. 当训练数据很小, $x$ 是一个很大的向量时, $x$ 的解可能有很多, 线性方程组欠约束有无穷解. 通过 L1 正则化缩小解空间, 使大部分分量为 0

为什么损失函数中加入 L1 范数会给出稀疏解? L1 范数的形状, 他的尖峰恰好是稀疏解, 该点用来接触解向量所在曲面很可能在尖端找到一个交点, 从而得到稀疏解

  1. L2 正则化. 添加 L2 范数, L2 正则化倾向于让权重值变得较小, 但不会导致权重稀疏, 不具备特征选择的作用, 可以有效控制模型复杂度. 类似上图, 当方形变为圆, 可以看到不具备特征选择作用.
L2 正则化使权值尽可能小, 参数值小的模型比较简单, 能适应不同的数据集, 在一定程度上避免了过拟合现象. 若参数很大, 那数据只要偏移一点点也会对结果有很大影响

为什么 L2 正则化能获得很小的参数? 如下图, 回忆一下高中知识, 两个圆心间直线距离最短, 也即经过相切的点距离最短, 此时参数最小

  5. Dropout

    随机删除一部分隐式神经元, 如下图, 虚线连接的单元不更新. Dropout 的训练过程相当于训练了很多只有部分隐层单元的神经网络, 每个网络都有一个分类结果, 大部分网络都可以给出正确的分类结果, 那么少数的错误分裂就没有很大影响

  6. Batch Normalization

    BN 一般加在非线性层前面, BN 作用:

    1. 加速收敛. 让数据分布本身就在 0 均值处

      输入的分布: 网络更新一次, 网络后面的层输入数据的分布会发生较大变化, 此时训练很慢不一定收敛. 如果每一层的输入相对稳定, 收敛就会更快. 白化也即将数据变为均值为0, 方差为1的高斯分布, 可以加快网络收敛, 但白化复杂费时, 退而求其次改为将输入做移动和线性变换, 使输入均值为0, 方差为1

      系数 r, 偏置 b: 单纯的归一化后, BN 的输出更可能在激活函数的线性段, 多个线性层可以用一个线性层来代替, 这样网络的非线性表达能力差. 所以加上系数 r 和偏置 b 让输入能达到激活函数的非线性区域

    2. 避免过拟合

      样本间有了交互, 对于同一个训练样本, 网络的输出会发生变化, 提升了网络的泛化能力

    3. 降低网络对初始化权重的敏感度. 如 a.

    4. 允许使用较大的学习率

    训练时有 batch, 测试时 bs=1, 无法正常使用 BN. 所以网络测试时 BN 层的均值和方差使用的是训练时所有 minibatch 对整个数据集的均值方差的无偏估计

  7. 残差结构

    网络过深, 优点是得到高阶语义信息, 缺点是信息损失, 可能产生梯度损失或爆炸

ROC 曲线

二分类问题可以通过调整分类器的灵敏度得到不同的分类结果, 从而在二者间折中. 将各种灵敏度下的指标连成曲线可以得到 ROC 曲线, 可以更全面地反映算法性能

真阳率 TPR 即召回率: 正样本被判正的比例

TPR=TPTP+FNTPR = \frac{TP}{TP+FN} TPR=TP+FNTP​

假阳率 FPR 即误报率: 负样本被判正的比例:

FPR=FPFP+TNFPR=\frac{FP}{FP+TN} FPR=FP+TNFP​

精度, 召回率, F1 值

TP: 正样本被判正

FN: 负样本被判正

TN: 负样本被判负

FP: 正样本被判负

精度: 真正的正样本所占比例

P=TPTP+FPP = \frac{TP}{TP+FP} P=TP+FPTP​

召回率: 被判正的正样本占所有正样本的比例, 越高查的越全:

R=TPTP+FNR = \frac{TP}{TP+FN} R=TP+FNTP​

F1 值: 精度和召回率的调和平均值的倒数, 值越大越准确:

1F1=12(1P+1R)\frac{1}{F1} = \frac{1}{2}(\frac{1}{P}+\frac{1}{R}) F11​=21​(P1​+R1​)

交叉验证

用于统计模型精度

原理: k 折交叉验证将样本随机均匀地分为 k 份, 轮流用其中 k-1 份训练模型, 1 份用于测试模型准确率, 用 k 个准确率的均值作为最终的准确率

奥卡姆剃刀原则

简单的模型通常具有更好的泛化性能
